# 📝 To-Do App 💖

A simple and cute *To-Do List application* built using *Python* and *Tkinter*.  
Keep track of your tasks in a fun and interactive way! 🌸✨

## ✨ Features
- Add new tasks  
- Edit existing tasks  
- Delete tasks when done  
- User-friendly GUI 💻

## 🛠 Built With
- Python  
- Tkinter
